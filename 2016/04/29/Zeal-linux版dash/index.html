Zeal-linux版dash

发表于 2016-04-29   |   分类于 linux   |  

ZEAL 是一款离线文档浏览器,其灵感来自 OS X平台上的 Dash，目前支持 Window 和 Liunx。基于 QT5。

主要功能：

  • 可同时搜索多个文档
  • 不依赖网络
  • GPL 协议开放源码
  • Dash 中的文档都可以在 Zeal 中使用。

安装

到zeal的官网download页面，按照对应系统的方式安装

我用的是linux mint ,也就是按照ubuntu的安装方法安装：

使用

安装完成后，到File–>Options–>Docsets中浏览下载你想要的文档。
下载后就可以搜索查找你想要的：
eg:

string ：将会查找所有关于string的文档
python:string： 将会查找与python有关的string
