一些好用的插件

  • Package Control

    用来安装其它插件，安装方法可以去Package Control查看，注意Sublime Text的版本问题。

  • Emmet

    前端神器

  • FileHeader

    自动创建文件开头模板，并且会根据最后的保存时间修改更新时间

  • AllAutocomplete

    Sublime Text 默认的 Autocomplete 功能只考虑当前的文件，而 AllAutocomplete 搜索所有打开的文件来寻找匹配的词。

  • Terminal

    打开在当前文件所在的目录的终端，默认情况下，它设置按 Ctrl / Cmd + Shift + T 键的快捷方式打开终端。这也是打开上次关闭的文件的快捷键，你需要修改快捷键来兼容两个功能
    等等还有很多好用的插件，可以自行搜索
