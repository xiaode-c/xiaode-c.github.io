ubuntu安装node

node.js nrm gdb
2016-3-17

学习一下node.js，感受一下不同的编程风格，我用的是ubuntu系统。

安装版本管理器

安装node.js版本管理工具，这里我选择的是n

    git clone https://github.com/tj/n.git
    cd n
    make install

安装node.js

    n + (node.js的版本号)
    eg: n 4.2.1

安装nrm

安装nrm,npm源的管理工具可以很方便的切换源，很适合在国内这种情况下使用。

    npm install -g nrm
    
    #显示可选的源
    nrm ls
    
    #测式各个npm的速度
    nrm test
    
    #切换源
    eg:
      nrm use taobao
