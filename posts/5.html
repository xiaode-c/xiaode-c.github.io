在sae上部署应用

首先新建一个应用（python为例）,然后使用git或者svn工具来创建一个版本假设您刚刚创建了应用newapp。

创建应用

首先，创建一个新的Git仓库并且添加一个Git远程仓库sae，地址为：https://git.sinacloud.com/newapp。

mkdir newapp && cd newapp
git init
git remote add sae https://git.sinacloud.com/newapp
